## Getting started

> **Important!** requires Node >= 14, Express >= 4 and TypeScript >= 4.

```batch
# install dependencies
$  yarn

# serve
$  yarn start

# build for production
$  build
$  start:prod
```

## Docker

> By Default Port :: 8083

```
# Make sure The project is built before using Docker
$ yarn build

# build docker image
$ docker compose build

# start docker image
$ docker compose up
```
